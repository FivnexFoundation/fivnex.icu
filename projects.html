Name Status Latest Stable Latest Current Description

HTOS Active N/A N/A Love the internet but use nothing else on your system? HTOS is a browser-only operating system using the Tor web browser.
Hyperkernel Active N/A N/A A kernel-only operating system where everything is a kernel module, or directly built into the kernel.
KTROS Development Paused N/A N/A One programming language. Nothing else. Whole OS built with Fivnex's Keter programming language.

VMOS Active N/A N/A Ever needed to run virtual machines, but absolutely nothing else? Us neither, but here is a solution to that anyways!
VSCOS Active N/A N/A Visual Studio Code as the entire graphical interface. Terminal, file system reader, document editor, and everything. No other apps, no other GUI's (other than a web browser).
